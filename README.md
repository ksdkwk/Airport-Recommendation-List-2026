# 1. 先说一下为什么不推荐使用免费机场
因为免费机场几乎完全不安全（免费机场几乎不会做可信加密），风险远大于你使用免费额度的收益。所以这里完全不推荐你使用免费机场
	1.1 全程嗅探浏览数据：服务商可以看到你所有访问记录、网页表单、账号、Cookie。如果你登录社交平台、邮箱、网银，账号极易被窃取。
	1.2 中间人劫持、篡改网页：可能偷偷植入广告、钓鱼代码，把正常网站跳转成诈骗页面；部分恶意节点会窃取上传文件、照片、文档。
	1.3 出售你的浏览日志牟利：免费运营方主要收入：打包售卖用户上网行为记录、画像给广告商、灰色渠道。
	1.4 暗藏后门、木马：不少免费订阅、客户端捆绑恶意程序，读取手机 / 电脑本地文件、通讯录。
==通俗理解：付费机场你是客户；免费机场，你本身就是商品。==
此外，凡是机场都有停服/跑路的风险，因此不建议你订阅大额年付，而是进行**月付**或者是可以*购买的不限时定量额度*。
